# Seche

Grid system for Rails Applications

## Installation

Add the seche gem line to group called :assets in your application's Gemfile:

    group :assets do
      gem 'sass-rails'
      gem 'seche'
    end

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install seche

## Usage

  in sass/scss file:
  @import seche
